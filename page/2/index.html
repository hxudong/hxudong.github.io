没有天，只有自己

By 旭东

发表于 2017-07-02

高三的时候凯伟说他有两句口头禅，一句叫“人的命，天注定”，另外一句是“事在人为”，刚开始他用这两句话是想来表明他自己的矛盾心理的，但是现在我想说，其实这两句话并不是矛盾的，他可以合并成一句话。

一个人的命，通过这个人都做了些什么，而确定的。天是”确定”的主体，人是”确定”的客体。天通过规律来确定一个人的命。以前我曾经说过类似的一句话，“一切归诸于天命之事必有其人事之原因”。意思也就是说我们不要寄希望于天，更不要寄希望于人，而要寄希望于自己，寄希望于自己所做的事。不怨天尤人。马克思已经说过“物质世界只能通过物质力量来改变”（1）。所以，我们不能期望着天上掉馅饼这回事。

“哲学家们只是用不同的方式解释世界,而问题在于改变世界”（2）。

我希望人们看到这句话的时候有一种心惊肉跳，激动人心的感觉。自此我们便真的明白了自己的命运到底由谁来决定，自己怎样来改变自己的命运。从此我们在思想上能够首先解放出来，我们是不依靠于他人的，我们是自主的，并且可以利用自己的这份自主来完成更重要的事情.

是以我们得发挥自己的主观能动性，有很多人曾经，现在，还有将来会存在于这个世界上，平常我们都在抱怨自己的生活怎么样，怎么样，被程式化掉了，但是作为一个独立的个体，我们是否又做出过努力，来显示自己独特的人格，以及作为一个人，一个与众不同的人的超凡之处，直到今天，辫子掉了，但是人的封建的人身依附关系观念依然存在，人自己并不独立。
所以人自己需要有“独立之人格，自由之思想”。
人之于动物其不同之处，在于主观能动性，我们可以去按照自己的思想去做一些事情。只有做了事情，客观物质世界才会改变，其后，主观的思想能够做出反映，意识受到影响。在这之中，反思，这一个东西，是必不可少的。这样才可以不断的做出调整。

天，这一古老的提法，神秘，未知，不可测。是人对这些东西不可解释而做出的一种无可奈何的解释罢了。

多少年来我们一直在实事求是，解放思想的思想路线上走着。实事求是，一切从实际出发，这并没有错，认清现实是解决一切问题的关键。（这里暂时不说认识论的问题。）解决问题在这里应该怎么办？

这里主要写两种常犯的错误———-经验主义与教条主义。

教条主义、经验主义从形式上看是对立的，但是二者产生的根源是相同的，实质上都是唯心主义在实际工作中的具体表现。二者的出发点是相同的，都是从主观出发，行为上都是不问具体情况，不从实际出发，主观认识脱离当时当地的客观实际，一刀切，瞎指挥，都是主观主义的办事和思想方法。二者的结果相同，都必然会带来损失甚至失败。

尊重规律，而不是囿于常规，经验主义错就错在了把常规当成了规律，把以前成功过的一次很多次的事物只要认为符合这种东西，就必然产生那样的结果。把偶然性当成了必然性。即使是有必然性在内的东西，又不考虑此次当下这一具体事件具体人物的特殊性。这样并不是说经验没有用，每次都是新的，规律就是一坨翔而已，只要我眼下好好弄就好了。认识了特殊性的“性”之所在。对于此“性”的问题的时候正是规律，经验发挥作用的时候。擦亮眼睛正是此处所在。这个世界不是我们创造的，我们也不是神，是人。老祖宗的智慧是要学习的。人是全新的，慢慢的会变成旧的，希望我们变旧的同时，这个世界变新了，这也算是我们做出的一份贡献吧，没有白来这一趟。

批教条主义，大家可能第一个要批得就是我了，我这么爱讲道理，明显就是一个教条主义者，一切按着书上的来，哈哈。“不唯上，不畏权，不囿于已成之理”这就反教条主义，我的固执来源于我的不理性，世界上本来就不是一个纯理性的，人的热血的，人是有感情的动物，把我生命中美好的东西让我剔除掉，让我变得不再完整，大家就开心了，我不开心，所以我不会那么做，心情，是心的情绪，除了理性还有其他两样东西的，无论别人是否相信，我愿意相信，我会一直相信，人不吃饭，身体会死，人不相信，对什么都在乎，心就死了。如果人没有灵魂，那活着有什么意思？如果人有灵魂，那要这幅躯壳又什么用？

“宠辱不惊，闲看庭前花开花落，去留随意，慢看天际云卷云舒”，没有什么是必然的，只有我一直在是必然的。渡尽劫波之后，不仅仅有兄弟在，姐妹也在，那些年的那些人也在，大家都在。保持一个好心情，对自己身边的人好，让他们开心，让自己开心。丘吉尔有这么一句话，“有可能坐着，我就不站着，有可能躺着，我就不坐着”翻版一下，只要能让别人和自己一般，就不要伤心，难过与无奈；只要能让别人和我笑，我就绝对不再板着脸装一般，漠不关心是我最最最痛恨的事情。世界凉了，心还有温度就说明还活着，我还在。

“天地不仁，以万物为刍狗”。

闲来扯淡

By 旭东

发表于 2016-12-24

看这题目，肯定以为我是一个闲人，因为只有闲人才能把时间拿出来扯蛋，其他人哪来的时间扯淡啊。其实不对，我不是个闲人，而且更没时间来扯淡。
樱花节都过去那么长时间了，记得的也没有很多了。印象最深刻的就算是来武大看樱花的人比较多。其实，花有百种，好看的不只只有这樱花，就我知道的而言，比如梅花，朵朵花开淡墨痕，迎春花，二月迎春花盛柳，清香满串荡悠悠，以及儿童急走追黄蝶，飞入菜花无处寻的油菜花、其他的如山茶，白玉兰，海棠，紫荆花等都是极有看透的，春天是百花盛开的季节，其他的花也开了，但是丝毫也没有为武大的樱花减轻点压力。再说这樱花，武大的，其他的学校也有啊，近的不说，武汉各个学校都有那么几株，远的像西交大的樱花也是极有名气的，年年人家还要弄一个樱花祭呢，但是也没有武大这么倒霉，这么人多，一个人20的门票也挡不住，每天几万甚至十万的游客，搞得好像武大的樱花比交大的好看似的。

一句老话说的好“外国的月亮比中国圆”，我觉得月亮其实是一样圆的，武大就是赏樱的氛围好一些，大家都经常去那里看，慢慢的人也就多了起来，渐渐地也就形成了一个习惯，每年都去那里玩一玩。鲁迅周先生说中国人有个特点就是“捧与挖”。觉得武大的樱花就是被捧上去的，被捧到一种好像不来看看武大樱花就像没到过武大，甚至就像没看到过真正的樱花似的（说实话武大的确实有好多种樱花，什么云南的，日本的，什么什么的，各种都有），譬如去北京没到过天安门，去巴黎没看到艾菲尔铁塔，伦敦没看到大笨钟，纽约没看到帝，国大厦一样。其实哪有那么厉害，北京不也有故宫吗？其他的几个大城市也有其他的东西，可是人们的眼睛往往是定在那个感觉最厉害的上面，其他的被忽略掉，就像那个笑话里面，你知道世界第一高珠穆朗玛峰，那第二呢？第三呢？多委屈啊他们俩，就这么被人给比下去了。

作为一个武大的孩子，有人这么喜欢武大，喜欢武大的樱花，我当然高兴，可是有那么多的人仅仅是喜欢曾经去过武大，曾经看过武大的樱花而已，来了武大，买点东西，照张相，然后写到此一游已经不时髦了，就干点别的。喜欢武大就别在武大里面乱搞，但是游客的行为实在是令人发指啊，当然现在好多了。（不能这么说同胞，多不和谐）武大不是景点，门票也不是你可以在武大里面恣意妄为提前付的账单。

说远了，其实要说武大的花，我觉得都挺好的，尤其是那些安安静静的花，就那么开着，享受阳光，沐浴春风，多么舒服的一件事，我在路上曾经见过两种给我很有感觉的，一种是在宿舍旁边草丛中看到的蓝色的，如一条小河在碧绿的草原上流淌而过，静水流深。又如微风中的蒲公英，轻巧、活泼，给人一种好像人在笑的感觉；一种是在主教下面那种黄色的小花，淡雅而又灿烂、纯洁而又富有生气，花中透着一股灵气，直逼人的眼，真觉得可以净化人的灵魂一般。

为了不打搅她们，当时也没有拍照，觉得也是怪可惜的，想一想也许他们想要的就是那么一份安静与懂的罢了，这么一想也就不怎么遗憾了。

忙得没有效率，很久也没有联系老同学

By 旭东

发表于 2016-11-05

按道理来讲，现在的我应该做项目才对，因为手里头现在有很多事积压下来了，应该赶快弄完，再去做别的才是，但是今天或者是远一点的一段时期内，感觉自己做了很多杂事 ，所以做的事都是没有效率和意义的，因为自己一无所获，从中没有任何的学习。都是为了去完成而做的，感觉这些其实也都是可以避免的，但是却都是没有避免。

现在有一个想法就是不太爱热闹了，有点开始喜欢安静。可能是以前就喜欢安静，只是这么明确而清晰地感受到了自己的心意。静极思动，动极思静。前天去试验场做试验，也算是晒了一天的太阳。中午看到工操的草坪上躺了个人，在那躺着晒太阳。太舒服了。以前自己上高中的时候和凯伟有一次在新建好的草坪上躺了一会儿。周围的人就有点笑话了，而我和他竟然还真的有点在乎他们的笑话了。现在想来那个时候大家还不是都有一个宽容自由的思想，都被所谓的社会规则舒服了很多，自己当时还是意志不坚定，也是受到他们言语的影响。

人呢，真是人生生而不自由，要受到那么多玩意的限制。在合理的范围内，我觉得还是得更要去主动一点。不能“己所不欲，勿施于人”，也不要“己所欲，施于人”。善于拒绝有时候会更好的保持自己的独立人格以及思想与行动上的自由。因为脑子里以及整个人身体被自己内心认为没意义的事情充斥的时候真实麻烦，甚至讨厌。

有好几个老同学本来说，要忙完这一段打个电话好好聊一下的，所以就更讨厌了。

已经有很多年，或者说具体的讲应该是有两年，没有和人再辨过，因为第一华佗和我已经不再停留在“理论辩论”的阶段，每次两个人都是就所谓的世界观，人生观，价值观，怎么样去做事，做人进行一次争论，然后找到一个答案，找出一个自己所谓的正确观念，现在两个人一个人在深圳正在工作为生活努力，一个人正在所谓的实验室里整天所谓的忙，都在实际生活中实践着自己的信念，然而怕就怕在这样的过程中渐渐得都忘掉了自己是为什么而努力，只有忙，没有生活。第二则是，华佗和我不讨论以后，也没有人和我进行这方面的交流了。

我想那才是我22年过得最开心，最有意义的时光了。我想你们了。

自强才能不息

By 旭东

发表于 2016-09-11

##技术还是掌握在自己的手里好
学习的过程中，可能会有各种各样的行业内的知识，工科尤其是有许多相关的软件更是极其多，然而，这种东西，随着整个的科研氛围趋向于保守，同行之间作为“冤家”，当然是不怎么愿意讲自己所学交给你，这种时候自然就会体验到“独立自主”这几个字的重要性，所以说，要是遇到助你的贵人，一定是要感谢一下。

天行健，君子以自强不息。以前可能是背着装逼，没有去深究这句话后面的意思，其实这句话已经很好理解，只要用字面意思就可以了。君子所以不息着，以期自强而已，自强。

马克思以前说过“哲学家们总是将重点去认识世界，而重点应该是去改造世界”，所以这些道理什么的，如果不转化为行动，就根本显示不出其力量，可以改造一个人，改造世界的力量。
“物质世界只有用物质力量才能改变”，所以当想到什么的时候，要立刻随手再想到一套实施方案，而不是先发个博客，博客应该做完以后再写，作为总结。
立刻动手起来。千里之行，始于足下，先走一步，走一步，再走一步。
虽然说要互相关爱，但是还是更要独立更生。
