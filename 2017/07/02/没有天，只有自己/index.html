没有天，只有自己

高三的时候凯伟说他有两句口头禅，一句叫“人的命，天注定”，另外一句是“事在人为”，刚开始他用这两句话是想来表明他自己的矛盾心理的，但是现在我想说，其实这两句话并不是矛盾的，他可以合并成一句话。

一个人的命，通过这个人都做了些什么，而确定的。天是”确定”的主体，人是”确定”的客体。天通过规律来确定一个人的命。以前我曾经说过类似的一句话，“一切归诸于天命之事必有其人事之原因”。意思也就是说我们不要寄希望于天，更不要寄希望于人，而要寄希望于自己，寄希望于自己所做的事。不怨天尤人。马克思已经说过“物质世界只能通过物质力量来改变”（1）。所以，我们不能期望着天上掉馅饼这回事。

“哲学家们只是用不同的方式解释世界,而问题在于改变世界”（2）。

我希望人们看到这句话的时候有一种心惊肉跳，激动人心的感觉。自此我们便真的明白了自己的命运到底由谁来决定，自己怎样来改变自己的命运。从此我们在思想上能够首先解放出来，我们是不依靠于他人的，我们是自主的，并且可以利用自己的这份自主来完成更重要的事情.

是以我们得发挥自己的主观能动性，有很多人曾经，现在，还有将来会存在于这个世界上，平常我们都在抱怨自己的生活怎么样，怎么样，被程式化掉了，但是作为一个独立的个体，我们是否又做出过努力，来显示自己独特的人格，以及作为一个人，一个与众不同的人的超凡之处，直到今天，辫子掉了，但是人的封建的人身依附关系观念依然存在，人自己并不独立。
所以人自己需要有“独立之人格，自由之思想”。
人之于动物其不同之处，在于主观能动性，我们可以去按照自己的思想去做一些事情。只有做了事情，客观物质世界才会改变，其后，主观的思想能够做出反映，意识受到影响。在这之中，反思，这一个东西，是必不可少的。这样才可以不断的做出调整。

天，这一古老的提法，神秘，未知，不可测。是人对这些东西不可解释而做出的一种无可奈何的解释罢了。

多少年来我们一直在实事求是，解放思想的思想路线上走着。实事求是，一切从实际出发，这并没有错，认清现实是解决一切问题的关键。（这里暂时不说认识论的问题。）解决问题在这里应该怎么办？

这里主要写两种常犯的错误———-经验主义与教条主义。

教条主义、经验主义从形式上看是对立的，但是二者产生的根源是相同的，实质上都是唯心主义在实际工作中的具体表现。二者的出发点是相同的，都是从主观出发，行为上都是不问具体情况，不从实际出发，主观认识脱离当时当地的客观实际，一刀切，瞎指挥，都是主观主义的办事和思想方法。二者的结果相同，都必然会带来损失甚至失败。

尊重规律，而不是囿于常规，经验主义错就错在了把常规当成了规律，把以前成功过的一次很多次的事物只要认为符合这种东西，就必然产生那样的结果。把偶然性当成了必然性。即使是有必然性在内的东西，又不考虑此次当下这一具体事件具体人物的特殊性。这样并不是说经验没有用，每次都是新的，规律就是一坨翔而已，只要我眼下好好弄就好了。认识了特殊性的“性”之所在。对于此“性”的问题的时候正是规律，经验发挥作用的时候。擦亮眼睛正是此处所在。这个世界不是我们创造的，我们也不是神，是人。老祖宗的智慧是要学习的。人是全新的，慢慢的会变成旧的，希望我们变旧的同时，这个世界变新了，这也算是我们做出的一份贡献吧，没有白来这一趟。

批教条主义，大家可能第一个要批得就是我了，我这么爱讲道理，明显就是一个教条主义者，一切按着书上的来，哈哈。“不唯上，不畏权，不囿于已成之理”这就反教条主义，我的固执来源于我的不理性，世界上本来就不是一个纯理性的，人的热血的，人是有感情的动物，把我生命中美好的东西让我剔除掉，让我变得不再完整，大家就开心了，我不开心，所以我不会那么做，心情，是心的情绪，除了理性还有其他两样东西的，无论别人是否相信，我愿意相信，我会一直相信，人不吃饭，身体会死，人不相信，对什么都在乎，心就死了。如果人没有灵魂，那活着有什么意思？如果人有灵魂，那要这幅躯壳又什么用？

“宠辱不惊，闲看庭前花开花落，去留随意，慢看天际云卷云舒”，没有什么是必然的，只有我一直在是必然的。渡尽劫波之后，不仅仅有兄弟在，姐妹也在，那些年的那些人也在，大家都在。保持一个好心情，对自己身边的人好，让他们开心，让自己开心。丘吉尔有这么一句话，“有可能坐着，我就不站着，有可能躺着，我就不坐着”翻版一下，只要能让别人和自己一般，就不要伤心，难过与无奈；只要能让别人和我笑，我就绝对不再板着脸装一般，漠不关心是我最最最痛恨的事情。世界凉了，心还有温度就说明还活着，我还在。

“天地不仁，以万物为刍狗”。
